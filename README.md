# Brand Assets Manager

## The Problem

Design teams waste countless hours searching for the right assets across scattered folders, outdated design files, and endless Slack threads. Brand inconsistency creeps in when developers can't find the correct logo variant, designers use deprecated color palettes, and marketers pull outdated typography specifications. **The average team member spends 3-5 hours per week just hunting for brand assets.**

## The Solution

Brand Assets Manager is a centralized, beautiful hub for all your design system assets. Built for modern teams who value consistency and speed, it provides instant access to your typography, colors, gradients, logos, icons, and UI componentsâ€”all in one elegant interface.

## ðŸš€ Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/irtizaaftabmian/Brand-assets.git

# Navigate to the project
cd Brand-assets

# Install dependencies
npm install

# Start the development server
npm run dev
```

Visit `http://localhost:3000` and start building your design system hub!

### Build for Production

```bash
npm run build
```

Deploy the `build` folder to any static hosting service (Vercel, Netlify, GitHub Pages, etc.)
