# Screenshots Guide

Please take the following screenshots and save them in this folder:

## Required Screenshots:

1. **light-mode.png**
   - Open the app at http://localhost:3000
   - Make sure you're in light mode (click the sun/moon icon if needed)
   - Show the main interface with some assets loaded
   - Recommended size: 1920x1080 or 1440x900

2. **dark-mode.png**
   - Switch to dark mode using the theme toggle
   - Show the same or similar view as light mode
   - Recommended size: 1920x1080 or 1440x900

3. **asset-management.png**
   - Show one of the asset tabs (Typography, Colors, Gradients, etc.)
   - Make sure some assets are visible
   - Capture the clean, organized layout
   - Recommended size: 1920x1080 or 1440x900

4. **share-feature.png**
   - Click the "Share All" button
   - Capture the share dialog showing the generated link
   - Recommended size: 1920x1080 or 1440x900

## Tips:
- Use full browser width for best results
- Make sure the UI is clean (no console errors visible)
- Capture at high resolution for clarity
- Consider using a screenshot tool like:
  - macOS: Cmd+Shift+4 (or Cmd+Shift+3 for full screen)
  - Windows: Win+Shift+S
  - Or browser extensions for full-page captures

Once you've added the screenshots, they'll automatically appear in the README!
